介紹
紀錄屬於自己的餐廳清單，可以瀏覽餐廳、查看詳細資訊、甚至連結到地圖。

功能
查看所有餐廳
瀏覽餐廳的詳細資訊
連結餐廳的地址到 Google 地圖
搜尋特定餐廳
開始使用
請先確認有安裝 node.js 與 npm

將專案 clone 到本地

在本地開啟之後，透過終端機進入資料夾，輸入：

npm install
安裝完畢後，繼續輸入：

npm run start
若看見此行訊息則代表順利運行，打開瀏覽器進入到以下網址

Listening on http://localhost:3000
若欲暫停使用

ctrl + c
開發工具

Express 4.17.1
Express-Handlebars 5.3.3
Bootstrap 5.2.1
